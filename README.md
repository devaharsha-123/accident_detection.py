# Accident Detection Using Machine Learning on Sensor Data

## Overview
This project is a machine learning-based accident detection system that predicts whether a vehicle accident has occurred using accelerometer sensor data. The model is trained on simulated sensor readings representing both normal driving conditions and accident scenarios.

The goal of this project is to demonstrate how structured sensor data can be used for real-time safety event detection through classification techniques.

## Features
- Simulates accelerometer sensor data for normal and accident conditions
- Trains a machine learning classification model using Random Forest
- Classifies new sensor readings as accident or non-accident
- Demonstrates a simple real-time prediction workflow
- Easy to understand and extend for further experimentation

## Tech Stack
- Python
- NumPy
- Pandas
- Scikit-learn
